\# Local Backup Snapshot Script (DevOps Mini Project)



This PowerShell script creates timestamped backup snapshots of a folder.



\## Features

\- Compresses source folder into ZIP snapshots

\- Names backups using timestamps

\- Stores backups in `snapshots/`

\- Optional retention policy (delete old backups)

\- Works fully on Windows without any tools

\# Notebook Backup Tool (DevOps Mini Project)



This Python script automatically backs up a folder into timestamped ZIP archives and keeps only the last N backups.



\## Features

\- Compresses files into ZIP

\- Stores backups in `/backups`

\- Deletes old backups (default: keep last 5)

\- Fully local, no dependencies



\## Run

```bash

python backup.py

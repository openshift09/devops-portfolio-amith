\# Sensitive Data Masker (DevOps Security Mini Project)



This script detects and masks sensitive data in log files:

\- Passwords

\- API Tokens

\- Secrets

\- Auth Keys

\- DB Passwords



\## Features

\- Masks sensitive fields (passwords/tokens)

\- Supports multiple patterns

\- Saves sanitized logs in `/reports`

\- Fully local, no external dependencies



\## Run

```bash

python masker.py

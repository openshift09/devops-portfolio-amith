\# HTTP Header Fetcher (DevOps Mini Project)



This Python script fetches the HTTP response headers from any URL and saves them to a detailed report.



\## Features

\- Checks status code

\- Fetches all HTTP headers

\- Saves results to `/reports`

\- Handles invalid URLs \& timeouts

\- Fully local, uses Python only



\## Run

```bash

python fetcher.py

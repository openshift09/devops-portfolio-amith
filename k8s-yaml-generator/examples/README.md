\# Kubernetes YAML Generator \& Validator



This project generates Kubernetes Deployment and Service YAML files using Python templates and validates them with custom rules.



\### Features

✔ Auto-generate Kubernetes Yamls

✔ Enforce DevOps best practices

✔ Validate YAML syntax

✔ Check for labels, probes, and resource limits

✔ 100% runnable on VS Code

✔ No Kubernetes cluster needed



This demonstrates DevOps automation skills used in real teams.

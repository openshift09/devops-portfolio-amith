\# Disk Space Analyzer (DevOps Mini Project)



This Python tool analyzes free/used disk space across Windows drives and generates a detailed report.



\## Features

\- Scans multiple drives

\- Shows free, used, and total space

\- Warns for low disk space

\- Saves reports in `/reports`

\- Fully local, no WSL/Docker required



\## Run

```bash

python analyzer.py

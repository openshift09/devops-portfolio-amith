\# Simple Port Scanner (DevOps Mini Project)



This Python script scans a list of common ports on a target host and reports whether each port is open or closed.



\## Ports Scanned

\- 22 (SSH)

\- 80 (HTTP)

\- 443 (HTTPS)

\- 3306 (MySQL)

\- 8080 (Web Apps)



\## Run

```bash

python scanner.py

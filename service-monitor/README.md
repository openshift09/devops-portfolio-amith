\# Windows Service Monitor Automation (DevOps Mini Project)



This PowerShell script monitors a Windows service and automatically restarts it if it stops.



\## Features

\- Continuous monitoring of any Windows service

\- Automatic restart on service failure

\- Detailed logs saved in `logs/service\_monitor.log`

\- Fully local, requires no external tools
